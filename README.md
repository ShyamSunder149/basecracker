# Basecracker
A tool using bruteforce approach to crack bases of any encoding

### List of crackable Bases
- Base 16
- Base 32
- Base 36
- Base 45
- Base 58
- Base 62
- Base 64
- Base 85
- Base 91
- Base 32768
- Base 65536

Usage

```
go run main.go -h
```

Example

```
go run main.go -ct <your ciphertext here>
```
